## 🎯 四种布局模式

### 模式 A: 人物全屏 + 画中画 B-Roll (最常见)

**代表视频**: 001, 002, 006, 007, 008

```
┌─────────────────────────────────────────┐
│                                    [♡]  │
│                                    [💬] │
│      【人物区域 70%】                [➤]  │
│       全屏居中                           │
│                                         │
│    ┌─────────────────────┐              │
│    │   B-Roll 画中画 30%   │              │
│    │   (下方中央)         │              │
│    └─────────────────────┘              │
│  ════════ 字幕区域 ════════              │
└─────────────────────────────────────────┘
```

**特点**:
- 人物占 **70%**，全屏居中口播
- B-Roll 以画中画形式出现在 **下方中央**
- 人物始终可见，B-Roll 辅助说明

- 小窗位置：上方中央 / 右上角（圆形）/ 下方中央

## 📐 结构模式

```
Hook (0-10s) → 核心要点 x 2-3 → CTA (结尾)
```

| 部分 | 时长占比 | 作用 |
|------|----------|------|
| **开场 Hook** | 5-10% | 争议观点/数据/问题，快速抓住注意力 |
| **主体要点** | 80-85% | 2-3 个核心要点，每个配合 B-Roll |
| **结尾 CTA** | 5-10% | 引导关注/推荐延伸内容 |

## ⏱️ 节奏特征

### 视频时长与镜头分析

| 视频 | 时长 | 镜头切换 | 平均镜头时长 | 节奏类型 |
|------|------|----------|--------------|----------|
| 001 | 01:39 | 25次 | 6s | 中快 |
| 002 | 01:19 | 24次 | 3.2s | 快 |
| 003 | 02:14 | 4次 | 33.5s | 慢（白板型） |
| 004 | 02:36 | 35次 | 4.5s | 中 |
| 005 | 02:18 | 10次 | 14s | 中慢 |
| 006 | 04:50 | 28次 | 10.3s | 中 |
| 007 | 04:28 | 62次 | 6s | 快 |
| 008 | 06:51 | 152次 | 2.7s | 快 |

### 节奏规律
- **短视频 (1-2分钟)**: 平均镜头 **3-6秒**
- **中等视频 (2-5分钟)**: 平均镜头 **4-10秒**
- **长视频 (5分钟+)**: 平均镜头 **2-6秒** (需更快切换维持注意力)
- **白板/PPT型**: 平均镜头 **20-40秒** (内容驱动)

## ✅ AI 生成规则清单

### 布局模式选择
- [ ] **口播为主** → 模式 A (人物70%全屏 + B-Roll画中画30%)
- [ ] **教学演示** → 模式 B (素材70% + 人物画中画30%)
- [ ] **纯干货** → 模式 C (无人物，纯PPT/白板)
- [ ] **长视频** → 模式 D (灵活切换全屏↔画中画)

### 人物设置
- [ ] 真人面向镜头
- [ ] 深色衣服为主（黑色连帽卫衣常见）
- [ ] 可配眼镜（增加专业感）
- [ ] 背景：室内浅色墙面/书柜

### B-Roll 规则
- [ ] 口播提到人物/产品/概念 → 触发 B-Roll
- [ ] 默认画中画模式（人物可见）
- [ ] 屏幕演示/教学 → 人物缩小到右下/下方
- [ ] 转场用直切，不用过渡效果
- [ ] 单个 B-Roll 时长 2-8 秒

### 字幕规则
- [ ] 白字黑边，位于中央偏下
- [ ] 可选逐字高亮（黄色/蓝色）
- [ ] PPT型视频可用黄色字幕

### 关键词卡片
- [ ] 出现专业术语/核心概念时弹出
- [ ] 位置：中央/左上角
- [ ] 背景：黑色块/白色块/半透明
- [ ] 带数字编号时更清晰

### 节奏控制
- [ ] 短视频(1-2分钟): 镜头 3-6 秒
- [ ] 中等视频(2-5分钟): 镜头 4-10 秒
- [ ] 开场节奏快(Hook)
- [ ] 中段配合 B-Roll
- [ ] 结尾放缓(CTA)
